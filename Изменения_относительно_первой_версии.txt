Изменения относительно первой версии программы (тезисно)

1) Реализовано корректное сохранение выходного файла через кроссплатформенный путь:
   - имя формируется как *_modify.xlsx рядом с исходным файлом.

2) Добавлен явный вывод результата выполнения:
   - сообщение об успешном сохранении файла;
   - сообщение об успешном завершении работы программы.

3) Исправлена повторная обработка уже измененного файла:
   - если лист "new" уже существует, он удаляется и создается заново.

4) Доработан парсинг блоков ТНВ:
   - границы блоков определяются устойчиво по колонке ТНВ;
   - устранены пропуски строк из-за вариативной структуры исходных листов.

5) Реализована адаптация к двум форматам исходной таблицы:
   - вариант с колонками "МДП с ПА";
   - вариант без "МДП с ПА".

6) Исправлено сопоставление колонок критериев:
   - устранена путаница между критериями "МДП без ПА" и "АДП";
   - восстановлена корректная запись в целевые колонки листа "new".

7) Реализовано сворачивание (скрытие) колонок, связанных с "МДП с ПА":
   - колонки E, H, K скрываются, если в исходном файле нет "МДП с ПА".

8) Доработано формирование блока "Минимальное из:":
   - добавляется первой строкой только при нескольких условиях;
   - не добавляется избыточно, если условие одно.

9) Исправлено цветовое форматирование "Минимальное из:":
   - зеленым выделяется только текст "Минимальное из:";
   - номер пункта и остальной текст окрашиваются корректно.

10) Улучшен разбор нумерации условий:
    - устойчивое чтение формата "N) текст" через регулярное выражение.

11) Добавлена нормализация формул:
    - токен MIN приводится к min;
    - улучшена обработка операторов и пробелов в формульных строках.

12) Добавлена корректная обработка переносов строк из Excel:
    - поддержка декодирования "_x000A_" в реальный перенос строки.

13) Улучшено автоопределение ширины столбцов на листе "new":
    - автоподбор по содержимому;
    - сохранение верхних ограничений ширины по колонкам.

14) Добавлена фиксация верхней части листа "new":
    - закреплены строки 1 и 2 (шапка таблицы).

15) Добавлена подготовка листов к печати/PDF:
    - "Обшая информация о сечении" и "new" настроены под горизонтальную A4;
    - для "new" включено повторение шапки на каждом листе.

16) Реализована актуализация гиперссылок на листе "Обшая информация о сечении":
    - ссылки перенаправляются на соответствующие строки нового листа "new".

17) Доработана строка-заголовок схемы (розовая строка на "new"):
    - используется только первая смысловая строка названия;
    - выравнивание выполнено по левому краю;
    - высота строки рассчитывается устойчиво для читаемого отображения.

18) Возвращено штатное поведение примечаний:
    - сохранен автоматический подбор размера примечаний (AutoFit),
      как в исходной логике, для корректного отображения в Windows.

19) Исправлена обработка пути файла в Windows-консоли:
    - удаляются нулевые и служебные символы из входной строки;
    - поддержан запуск через аргументы командной строки (drag-and-drop на exe).

20) Подготовлены две поставки для Windows:
    - net8 self-contained (без установки .NET);
    - net472 framework-dependent (меньше размер, нужен установленный .NET Framework).
